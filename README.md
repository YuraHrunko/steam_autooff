# AutoOff (Таймер + Steam)

Приложение для Windows, которое может **автоматически выключить ПК**:

- по таймеру
- после завершения загрузок Steam (мониторинг папки `steamapps/downloading`)

---

Если браузер открывает файл вместо скачивания — нажмите правой кнопкой по ссылке и выберите «Сохранить ссылку как...».

## Возможности

- Минималистичный тёмный интерфейс
- Две вкладки:
  - **Таймер**
  - **Steam**
- Блокировка переключения вкладок во время активного автовыключения
- Кастомное окно без системной рамки
- Поддержка Alt+Tab и панели задач
- Кнопка отмены в каждом режиме

---

## Как использовать

## 1) Режим «Таймер»

1. Откройте вкладку **Таймер**
2. Укажите часы, минуты, секунды
3. Нажмите **СТАРТ**
4. Для остановки нажмите **ОТМЕНА**

Когда время выйдет — ПК выключится автоматически.

## 2) Режим «Steam»

1. Откройте вкладку **Steam**
2. Нажмите **СТАРТ**
3. Программа начнёт проверять загрузки Steam
4. Когда загрузки закончатся, начнётся отсчёт до выключения
5. При необходимости нажмите **ОТМЕНА**

---

## Важные условия для Steam-режима

- Steam должен быть установлен
- Папка `steamapps/downloading` должна существовать
- Если Steam установлен в нестандартный путь — добавьте путь в список `STEAM_PATHS` в коде

---

## Запуск из исходников (опционально)

Требуется Python 3.10+.

1. Скачайте репозиторий
2. Запустите:

```bash
python autooff.py
```

или двойным кликом по `autooff.bat`.

---

## Сборка `.exe` самостоятельно

Используется PyInstaller:

```bash
pyinstaller --noconfirm --clean --onefile --windowed --name AutoOff --icon autooff_icon.ico autooff.py
```

Готовый файл появится в папке `dist/`.

---

## Структура проекта

- `autooff.py` — основная программа
- `autooff_icon.ico` — иконка для `.exe`
- `autooff.bat` — удобный запуск на Windows
- `dist/AutoOff.exe` — собранный исполняемый файл
